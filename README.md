# Welcome to your the BlueRobins React Native Variant !! :-)

This is an [Expo](https://expo.dev) project, which is a framework built on top of React Native

# Setup & Running

## Prerequisites
- [Node.js](https://nodejs.org/) (v18 or later)
- [Expo Go](https://expo.dev/client) app on your phone

## Installation

1. Clone the repo
```bash
   git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   cd YOUR_REPO_NAME
```

2. Install dependencies
```bash
   npm install
```

3. Start the dev server
```bash
   npx expo start
```

4. Scan the QR code in your terminal with:
   - **iOS** — Camera app
   - **Android** — Expo Go app
